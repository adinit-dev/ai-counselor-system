# AI Counselor System

AI-powered mental health screening platform for students.

Tech Stack:
- Frontend: HTML, CSS, JavaScript
- Backend: Flask
- Database: SQLite
- Visualization: Chart.js

Features:
- Student login and test assessment
- Counselor dashboard with risk analytics
- AI-based mental health risk detection
- Dynamic test creation

Status:
Frontend and backend completed.
Currently integrating AI prediction module.
